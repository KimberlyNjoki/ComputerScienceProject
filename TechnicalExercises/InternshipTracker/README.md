# Internship Tracker

Repository:
[https://github.com/KimberlyNjoki/internship_tracker]

Live Server:
[https://internship-tracker-six-alpha.vercel.app/]

Description:
A web application designed to help students organise internship applications, track deadlines and monitor application progress throughout the recruitment process. The platform provides a structured way to manage opportunities and stay on top of important application milestones.

Technologies Used:

* HTML5
* CSS3
* JavaScript


Key Features:

* Internship application tracking
* Deadline management
* Application status monitoring
* Organised opportunity management
* Responsive user interface
* Student-focused workflow design
