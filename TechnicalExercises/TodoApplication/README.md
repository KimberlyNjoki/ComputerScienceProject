# To-Do Application

Repository:
[https://github.com/KimberlyNjoki/todo_app]

Live Server:
[https://todo-app-beta-tawny.vercel.app/]

Description:
A responsive task management application developed to help users organise daily activities, track progress and improve productivity. The application allows users to create, manage and complete tasks through a simple and intuitive user interface.

Technologies Used:

* HTML5
* CSS3
* JavaScript

Key Features:

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Responsive design
* Clean and user-friendly interface
